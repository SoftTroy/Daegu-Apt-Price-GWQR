## 개요

본 프로젝트는 대구광역시 아파트 매매가격에 영향을 미치는 요인을 분석하고, 각 요인의 영향이 지역과 가격 수준에 따라 어떻게 달라지는지 확인하기 위해 진행했습니다.

일반 분위수 회귀분석(Quantile Regression), 지역 군집을 반영한 분위수 회귀분석, 지리적 가중 분위수 회귀분석(Geographically Weighted Quantile Regression)을 적용했습니다.

아파트의 전체 거래금액이 아닌 면적당 거래금액을 종속변수로 사용했습니다. 저가, 중간가, 고가 아파트 시장의 차이를 확인하기 위해 분위수 `0.25`, `0.50`, `0.75`에서 각각 모형을 추정했습니다.

## 분석 방법론

### A. 모델링 전략

* **Target:** 로그 변환한 면적당 거래금액
* **Quantile:** `0.25`, `0.50`, `0.75`
* **Location:** 아파트별 위도와 경도
* **Evaluation:** Check Loss

#### 1. Quantile Regression

전체 지역에 동일한 회귀계수를 적용하는 일반 분위수 회귀모형입니다.

평균적인 가격 변화를 추정하는 선형 회귀와 달리, 종속변수의 특정 분위수에서 설명변수의 영향을 추정합니다.

* `τ = 0.25`: 저가 구간
* `τ = 0.50`: 중위가격 구간
* `τ = 0.75`: 고가 구간

#### 2. Quantile Regression with Cluster

대구광역시 내부의 지역 차이를 반영하기 위해 군집 변수를 추가한 분위수 회귀모형입니다.

각 설명변수와 지역 군집의 상호작용항을 포함하여, 같은 변수가 지역에 따라 서로 다른 영향을 미치는지 확인했습니다.

#### 3. Geographically Weighted Quantile Regression

각 아파트의 위치를 기준으로 주변 관측치에 공간 가중치를 부여하고, 관측지점별로 분위수 회귀모형을 추정했습니다.

* **Distance:** 위도와 경도를 이용한 Haversine 거리
* **Kernel:** Adaptive Bi-square Kernel
* **Bandwidth:** 전체 관측치의 약 15%에 해당하는 최근접 이웃 수
* **Regression:** 가중 분위수 회귀분석

GWQR은 위치마다 별도의 회귀계수를 추정하므로, 아파트 가격 결정 요인의 공간적 차이를 확인할 수 있습니다.

### B. 데이터 처리

* 분석 범위를 벗어난 공간 이상치를 제거했습니다.
* 거래금액을 수치형 변수로 변환했습니다.
* 거래금액을 전용면적으로 나누어 면적당 거래금액을 계산했습니다.
* 건축연도를 이용해 건물연령 변수를 생성했습니다.
* 거래금액과 거리 관련 변수에 로그 변환을 적용했습니다.
* 위도와 경도를 제외한 분석 변수를 표준화했습니다.
* 학원 관련 변수는 주성분분석(PCA)을 이용해 축소했습니다.
* 첫 번째 주성분 점수에 `asinh` 변환을 적용했습니다.

## 성능 비교

세 모형의 성능은 분위수 회귀에 사용하는 Check Loss를 기준으로 비교했습니다.

* Quantile Regression

모델별 Check Loss는 다음 R 스크립트의 실행 결과에서 확인할 수 있습니다.

```text

```

GWQR로 추정한 지역별 회귀계수를 위도와 경도 좌표에 표시했습니다.

## 참고사항

GWQR은 각 관측지점에서 주변 데이터를 이용해 별도의 모형을 추정하므로 일반 분위수 회귀보다 계산량이 많습니다.

Adaptive Bandwidth가 지나치게 작으면 일부 지역에서 유효 표본이 부족해지거나 국지적 다중공선성 문제가 발생할 수 있습니다.

현재 코드는 전체 관측치의 약 15%를 최근접 이웃 수로 사용하도록 설정되어 있습니다.
